# Playwright Automation Testing Practice

A Playwright + TypeScript automation project demonstrating UI, API and End-to-End testing.

## Technologies
- Playwright
- TypeScript  
- GitHub Actions CI/CD

## Projects

### The Internet (herokuapp)
Practice automation testing for different webpage elements
- Dynamic Controls、Dynamic Loading
- File Upload、Checkboxes、Hovers
- Frames、Multiple Windows、Alerts
- Data Tables

### Restful Booker
Web application automation test
- **UI Tests**: Booking, Contact, Admin Login, Admin Room Management
- **API Tests**: Authentication, CRUD operations for Bookings
- **E2E Tests**: Customer make a booking → Admin check booking, Customer send message  → Admin verify message

## Run Tests
npm install
npx playwright test
