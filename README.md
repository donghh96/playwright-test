# Playwright Automation Testing Practice
This repository contains my Playwright learning projects, covering UI automation, API testing, and end-to-end testing using TypeScript.

## Technologies
- Playwright
- TypeScript  
- GitHub Actions CI/CD

## Projects

### The Internet
Practice UI automation for common web components and interactions.
- Dynamic Controls, Dynamic Loading
- File Upload, Checkboxes, Hovers
- Frames, Multiple Windows, Alerts
- Data Tables

### Restful Booker
Automation testing for a hotel booking web application.
- **UI Tests**: Booking, Contact, Admin Login, Admin Room Management
- **API Tests**: Authentication, CRUD operations for Bookings
- **E2E Tests**: Customer make a booking → Admin verifies the booking, Customer send message  → Admin verify message

## Run Tests
npm install
npx playwright test
